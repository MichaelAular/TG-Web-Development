Opdracht 1

Deze code werkt en complied naar:

.image { height: 800px; width: auto; }

Opdracht 2

Werkt niet. Er zijn niet dezelfde eenheden gebruikt. Code zou moeten zijn:

.image { height: 400px + 100px; width: auto; }

complied wordt dat dan:

.image { height: 500px; width: auto; }

Opdracht 3

Code werkt en complied naar:

.image { height: 100px; width: auto; }

Opdracht 4

Error! height wordt een negatief. Code moet zijn:

.image { height: 200px + 100px - 200px; width: auto; }

dat wordt dan:

.image { height: 100px; width: auto; }

Opdracht 5

werkt. resultaat wordt:

.image { height: 600px; width: auto; }

Opdracht 6

werkt niet! kan iets niet x 1px doen. code moet waarschijnlijk zijn:

.image { height: 750px * 1; width: auto; }

wat vertaalt in:

.image { height: 750px; width: auto; }

Opdracht 7

werkt eveneens niet. je kan ook niet delen door px. moet zijn:

.image { height: 30px / 5; width: 24px/ 4; }

wat in CSS komt te staan als:

.image { height: 6px; width: 6px; }

Opdracht 8

zie opdracht 7. Moet zijn:

.image { height: (30px / 5); width: (24px/ 4); }

wat vertaalt in:

.image { height: 6px; width: 6px; }
