Display & Positioning

OPDRACHT 1

de opdracht

Welke position values zijn er? Leg voor elke position values uit wat het doet. Geef ook aan welke default position waarde elementen hebben die je gebruikt in html.

de uitvoering

Er zijn 5 position values, te weten:

  • static
  • relative
  • fixed
  • absolute
  • sticky

static
Elementen met een static position worden niet beïnvloed door top, bottom, left of right properties. Dit is tevens de standaard positionwaarde gebruikt door HTML. Een elementen met position: static; wordt niet gepositioneerd op enige manier maar volgt de normale flow van de web pagina.

relative
Een element met position: relative; wordt gepositioneerd relatief aan zijn normale postitie. Met andere woorden: top, bottom, left of right properties doen het element verplaatsen relatief aan zijn oorspronkelijke positie. Andere content zal zich niet aanpassen om de gaten op te vullen door deze verschuiving.

fixed
Een fixed element is een element dat een vaste plek op het scherm heeft ongeacht eventueel scrollen. Het blijft dus altijd op de zelde plek staan. Top, bottom, left of right properties worden gebruikt om het element te postitioneren op het scherm. Een fixed element laat geen gat achter waar het oorspronkelijk had gestaan.

absolute
Een element met position: absolute; positioneerd zich relatief aan zijn eerste voorouder mét een postionering. Echter, zijn er geen voorouders met een position dan zal het element zich postioneren aan de hand van de body, wat betekent dat het met het scrollen van de pagina mee beweegt.
Let op: Een absolute positie haalt een element uit de normale flow van een pagina en kan dus over een ander element heen komen te staan!

sticky
Als laatst is er position: sticky;. Deze position is zowel relative als fixed, afhankelijk van hoe de gebruiker scrollt. In eerste instantie zal het element zich als een relative element gedragen totdat het een bepaalde positie bereikt op het scherm. Vanaf dan zal het element blijven plakken (sticky!) en op deze plek blijven staan.
