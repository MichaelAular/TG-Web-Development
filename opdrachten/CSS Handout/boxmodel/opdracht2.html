Het Box Model

OPDRACHT 2

de opdracht

Maak wat hieronder staat met paragraph en span tags.

de uitvoering

Dit is een block box en neemt de hele breedte in waardoor de volgende box op de volgende regel zit

In dit blok zit inline tekst in wat dus niet op de volgende regel komt te staan

De HTML hiervoor ziet er als volgt uit:

<div class="opdracht2"> <div class="block1"> Dit is een block box en neemt de hele breedte in ... </div> <div class="block2"> <br> </div> <div class="block3"> In dit blok zit <span>inline tekst</span> in wat ... </div> </div>

De CSS hierbij is:

.opdracht2 { max-width: 600px; border-style: solid; border-width: 2px; border-color: white; color: black; font-size: small; font-family: Georgia, 'Times New Roman', Time... } .opdracht2 span{ background-color: white; color: black; border-style: solid; border-width: 1px; border-color: black; } .block1 { display: block; background-color: #2a9e4d; border-style: solid; border-width: 1px; border-color: black; } .block2 { display: block; background-color: #63d184; } .block3 { display: block; background-color: #2a9e4d; border-style: solid; border-width: 1px; border-color: black; }
