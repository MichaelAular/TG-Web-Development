Het Box Model

OPDRACHT 5

de opdracht

Als je ruimte wilt maken tussen de border en content, welke css property gebruik je dan? Maak een voorbeeld in je antwoord.

de uitvoering

Om ruimte te maken tussen border en content gebruiken we padding. In een voorbeeld zal dat er als volgt uitzien:

de padding is blauw

De gebruikte HTML:

<div class="opdracht5"> <p>de padding is blauw</p> </div>

De gebruikte CSS:

.opdracht5 p{ color:white; background-color: black; text-align: center; border-style: solid; border-color: cyan; border-width: 1px; width: 400px; } .opdracht5 { padding: 10px; background-color: blue; border-style: solid; border-color: cyan; border-width: 1px; width: 420px; }
