Het Box Model

OPDRACHT 6

de opdracht

Als je ruimte wilt maken tussen de border en buiten de box, welke css property gebruik je dan? Maak een voorbeeld in je antwoord.

de uitvoering

Om ruimte te maken tussen border en de buitenbox gebruiken we margin. In een voorbeeld zal dat er als volgt uitzien:

element

Margin is uiteraard niet zichtbaar maar als we in de CSS kijken dan zien we dat er een marge is toegevoegd van 20px. Dit betekent dat er aan alle kanten (boven, onder, links en rechts) 20px afstand wordt gehouden van andere objecten.

De gebruikte HTML:

<div class="opdracht6"> <p>element</p> </div>

De gebruikte CSS:

.opdracht6 { border-style: solid; border-width: 1px; border-color: rgb(25, 82,76); background-color: rgb(29, 39, 38); color:turquoise; padding-left: 10px; padding-right: 10px; padding-top: 1px; padding-bottom:1px; width: 380px; text-align: center; margin: 20px; }
