Het Box Model

OPDRACHT 7

de opdracht

Maak nu een blok met width: 100px, height: 100px, margin: 5px, padding 5px en border-width: 5px. Zie hieronder:

  1. Hoe breed is de box?
  2. Hoe hoog is de box?
  3. Wat merk je op met width en height?

de uitvoering

Dit is de content

  1. De Box is 120px breed. (border 5 + padding 5 + width 100 + padding 5 + border 5 = 120)
  2. De Box is 120px hoog. (border 5 + padding 5 + width 100 + padding 5 + border 5 = 120)
  3. Width en heigt zijn van toepassing op het element. Padding, border en margin worden daar nog extra aan toegevoegd en vergroten hiermee de box.
