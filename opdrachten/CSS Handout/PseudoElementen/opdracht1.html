Pseudo Elements

OPDRACHT 1

de opdracht

Kun je enkele voorbeelden geven van pseudo-elementhtmlen (minimaal 4)? Beschrijf met je eigen woorden wat deze pseudo-elementen inhouden.

  • ::first-letter
  • ::first-line
  • ::before
  • ::after

::first-letter

Dit geeft een bepaalde stijl aan enkel de eerste letter van een tekst.

de HTML

<p>Let op de eerste letter!</p>

de CSS

::first-letter{ font-family:Algerian; font-size:3em; color: cyan; }

Voorbeeld

Let op de eerste letter!

::first-line

Dit geeft een bepaalde stijl aan enkel de eerste regel van een tekst.

de HTML

<p>Let op de eerste regel!<br> En de tweede.</p>

de CSS

::first-line{ font-family:Algerian; font-size:3em; color: cyan; }

Voorbeeld

Let op de eerste regel!
En de tweede.

::before

Dit geeft een bepaalde stijl voor een specifiek element.

de HTML

<ul type="none"> <li>Boter</li> <li>Kaas</li> <li>Eieren</li> </ul>

de CSS

li::before { content: " ♙ "; }

Voorbeeld

  • Boter
  • Kaas
  • Eieren

::after

Dit geeft een bepaalde stijl na een specifiek element.

de HTML

<ul type="none"> <li>Boter</li> <li>Kaas</li> <li>Eieren</li> </ul>

de CSS

li::after { content: " ♙ "; }

Voorbeeld

  • Boter
  • Kaas
  • Eieren
