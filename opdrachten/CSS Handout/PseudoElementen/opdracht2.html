::backdrop
dit pseudo element plaats een element ter groote van het gehele scherm achter de selector. Hierdoor kan je bijvoorbeeld de achtergrondkleur van een video op fullscreen aanpassen.

::cue
::cue spreekt VTT bestanden aan. Hierdoor kan je vormgeving van ondertiteling aanpassen.

::file-selector-button
Een ::file-selector-button wordt de button van de INPUT tag met het type="file". Hiermee kan je buttons maken om een bestand te uploaden.

::grammar-error
Hiermee spreek je tekst segmenten aan die de User Agent (bijvoorbeeld de browser) als gramaticaal incorrect bestempelt. Denk aan het rode lijntje onder een woord. Deze kan je nu aanpassen.

::marker
Een ::marker richt zich op de marker van een list item (meestal een nummer of een bulletpoint)

::part
Staat voor een element binnen een shadowtree met een gedeeld atribuut. BIjvoorbeeld een gedeelde DIV

::placeholder
Stijlt de placeholder tekst binnen een INPUT of TEXTAREA.

::selection
Als je het deel wat de gebruiker van je website selecteert wil stijlen kan je ::section gebruiken.

::slotted
Stijlt alle elementen binnen een SLOT.

::spelling-error Zie ::grammar-error

::target-text
Staat voor de tekst waar naartoe is gescrold (als de browser scroll-to-text ondersteunt).
