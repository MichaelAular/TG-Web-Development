Selectors

Er zijn verschillende soorten selectors te weten: type selectors, descendant selectors en class selectors. Hieronder krijg je uitgelegd wat het verschil is en wat je ermee kan.

Type Selectors

Een Type Selector, of ook wel een Element Selector, selecteren elementen met dezelfde “verbinding”. Denk hierbij aan span of paragraaf. De Type Selector wordt over het algemeen gebruikt om een site in grote lijnen aan te passen.

Het wordt vaak als een slechte gewoonte beschouwd om fijne detailwijzigingen toe te passen met alleen een Type Selector. Als je bijvoorbeeld een eigenschap "kleur: wit" op alle div-tags toepast, zou het zelden nuttig zijn op een site. Dit komt omdat div-tags generiek zijn en op sites en voor verschillende doeleinden worden gebruikt. Logischer is het om de Type Selector te gebruiken in de body en zo alle text in één keer wit te maken.

body { color: white; }

Descendant Selector

De Descendant Selector selecteert alle elementen die afstammelingen van een opgegeven element. Vaak hebben we het in HTML over parent en child. Een parent ligt als het ware een laagje hoger dan de child. Stel je hebt een div in die div staat een paragraaf gemaakt dan is de div de parent en de paragraaf daar een child in. De paragraaf stamt dus af van de div.

<div class="parent"> <p>dit is de child</p> </div>

Stel we willen voor de gehele website alle paragrafen binnen een div aanspreken kunnen we dit doen door gebruik te maken van de Descendant Selector. In het volgende voorbeeld worden alle p-elementen binnen div-elementen geselecteerd:

div p { background-color: yellow; }

Class selector

De .class selector selecteert elementen met een specifiek class attribuut.

Om elementen met een specifieke klasse te selecteren, schrijf je een punt (.) teken, gevolgd door de naam van de klasse.

.intro { background-color: yellow; }

Je kunt ook specificeren dat alleen specifieke HTML-elementen door een klasse moeten worden beïnvloed. Om dit te doen, begin je met de elementnaam en schrijft vervolgens het punt (.) teken, gevolgd door de naam van de class. zo wordt in het voorbeeld hieronder alle p elementen met class intro gestyled.

p.intro { background-color: yellow; }

HTML-elementen kunnen ook naar meer dan één klasse verwijzen.

<p class="intro large">2 classes!</p>
