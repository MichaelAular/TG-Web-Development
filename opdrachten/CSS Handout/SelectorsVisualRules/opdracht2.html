CSS Syntax

Syntaxis (van het Oudgriekse συν- syn-, "samen", en τάξις táxis, "regeling"), ook vaak zinsleer genoemd, is een deelgebied binnen de theoretische taalkunde dat in ruime zin de studie omvat van alles wat met de opbouw en structuur van zinsdelen en zinnen te maken heeft, ofwel van de zinsbouw.Net zoals bij de taal die je met andere mensen spreekt is er ook bij HTML en CSS sprake van een bepaalde syntax. Wat we hiermee bedoelen is dat elke taal aan bepaalde regels moet voldoen om begrijpbaar te zijn. Dit is niet anders voor een computer- of programmeertaal.

Een CSS code begint altijd met een selector. Dit is een altijd een HTML element of met andere woorden het onderdeel dat je gaat bewerken. Dit kan een div zijn, een paragraaf, H1, etc. Er zijn verschillende mogelijkheden. In het onderstaande voorbeeld zie je bijvoorbeeld een button als selector.

Hierna volgt een declaration block. Dit is het gedeelte waarbinnen één of meerdere declarations (statements) staan. Bij meerdere worden worden deze gescheiden door een puntkomma. Voor de leesbaarheid is het handig om elke declaration op een nieuwe pagina te beginnen. Dit blok wordt omsloten door accolades.

Elke declaration voldoet uiteraard ook weer aan een bepaalde syntax. In dit geval moeten we gaan benoemen wat we gaan bewerken (de property) en hoe (value). In het voorbeeld van de button gaan we twee dingen doen: We geven de button een rode kleur en een breedte van 50px. In de eerste declaration is de property dus color en de value red. In de tweede tweede is de property width en de value 100px. Let op! we scheiden deze twee altijd met een dubbele punt.

button { color: red; width: 100px; }
