OPDRACHT 1

Er zijn drie manieren om CSS (Cascading Style Sheets) in je HTML in te laden. Te weten: inline, interne en externe stylesheets Hieronder zal ik per onderdeel uitleggen wat dat inhoudt en waar je deze methode het best voor kan gebruiken.


Externe CSS

Met een externe stylesheet kan je door het wijzigen van één file de voorgeving van je gehele website aanpassen. Belangrijk is dat je deze file opslaat met een CSS extensie. Daarnaast moet je HTML file uiteraard weten van het bestaan van dit bestand en het daadwerkelijk gaan gebruiken. Dit bereik je door de bestanden te "linken" in je HTML File met het volgende stukje code:

<link rel="stylesheet" href="mystyle.css">

In dit voorbeeld (het CSS bestand is hier mystyle.css genoemd) en vertel ik het HTML bestand dat dit zijn stylesheet is. Dit stukje code plaats je in de head van je HTML file. LET OP: In een CSS bestand mag geen HTML staan!

INTERNE CSS

Een interne stylesheet kan gebruikt worden als een enkele pagina een afwijkende opmaak heeft. Hiervoor wordt de CSS code direct in het HTML bestand geplaatst. De CSS styling geldt hierdoor enkel voor deze pagina en zal bij aanpassing geen verdere gevolgen voor andere pagina's hebben. Let op! Deze code komt in de head van je HTML bestand te staan.

<head> <style> body { background-color: red; } h1 { color: blue; margin-left: 40px; } </style> </head>

In de head wordt middels een style command aangegeven dat het volgende stuk tekst over de styling gaat. Nu mag er in CSS geprogrammeerd worden! In dit geval veranderen we enkel voor deze pagina de achtergrondkleur naar rood en H1 naar blauw. Hierna sluiten het style command en kunnen we verer in HTML.

Inline CSS

Inline CSS als het kleinste stapje in de CSS matroushka. Hiermee kan je namelijk, zoals de benaming al doet vermoeden direct in een HTML regel een stuk styling aanpassen. Denk hierbij bijvoorbeeld aan het aanpassen van de styling van een enkele paragraaf (zie voorbeeld). Dit doe je door na het element (in dit geval de paragraaf) "style=" toe te voegen gevolgd door de gewenste styl. Let op! Dit gebeurt dus in de body van de HTML.

<body> <p style="color:red;">Enkel deze paragraaf is nu rood!</p> </body>
